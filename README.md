# Today_fashion

## サービス概要
Today Fashionとは、独自の服装指数を用いて、ユーザーの体質および気温に合わせて、
登録したファッションアイテムからおすすめのコーディネートを提供するサービスです。

## 想定されるユーザー層
- 気温や体質で服装選びを失敗したくない人
- 忙しい日常を送る学生や社会人
- ファッションに関心がある人

## サービスコンセプト
### 「服装指数の可視化」
Today Fashionは、多くのユーザーに向けて、当日の気温や各ユーザーの体質（暑がり・寒がりなど）を基準にした独自の服装指数を使用し、おすすめのコーディネートを提供するサービスです。

日本気象協会の服装指数では、気温に応じた服のアイコンが確認できるのみで、具体的なコーディネートの例は表示されません。また、競合アプリでは気温をもとに登録された服からコーディネートを提案しますが、ユーザーの体質が考慮されていない点が課題でした。

本サービスでは、アイコンを使ったコーディネート例の表示に加え、登録されたファッションアイテムの写真を用いた具体的なコーディネート例を提供します。

アイテムの管理・登録においては、簡単に登録できるよう、必須項目は「名前」「カテゴリー」「サブカテゴリー」の3項目のみとしました。さらに、写真をアップロードするだけで、OCRを使って画像を分析し、アイテム名を自動入力する機能も実装しています。より詳細に登録したいユーザー向けには、任意で複数の追加項目を設定しています。

私自身、平日・休日を問わず県をまたぐ移動が多く、移動先と住んでいる地域で気温差が大きい場合があります。寒暖差によって体調を崩すこともあるため、メイン画面では地域を2つ表示できるようにしました。また、旅行や出張など遠出の際には、行先の服装指数や5日間分のコーディネートを確認できるよう、日本地図を使った全国の服装指数予報機能も実装しています。

以上のことから
__服装選びを失敗したくない__ x __服装選びに時間を取られたくない__ x __クローゼットの可視化__ 

３点をメインターゲットとしてサービスを提供したいと思います。

## 実装している機能
### トップページ
- 本サービスについて概要
- ユーザー登録
- ログイン

### 天気と服装指数表示機能
- アカウント登録もしくはログイン後、設定した都道府県の天気や服装指数を表示し、ファッションアイテムの登録や、各都道府県の天気や服装指数を検索できる機能を表示
- 服装指数について、独自の設定によって調整する
（提案される服装は、体質設定および気温によって調整されます（例えば、登録した都道府県の気温が６度未満で体感設定が寒がりと設定したとき、ダウンコートをお勧めをし、体感設定が暑がりに設定したときはダウンコートではなくチェスターコートをお勧めするなど。今後はユーザーのフィードバックを取り入れて調整を行う予定です。）
- コーディネートの表現部分はデフォルトで用意したアイコンにて表現する。別途同じ表示画面に、登録した服がそのカテゴリーに登録してあれば追加で表示する。(カテゴリーにダウンコートが登録してあればその写真を表示させる)
- OpenWeatherMapを導入して、天気や気温情報を取得します

### 5日間の天気と服装指数表示機能
- 各都道府県の3時間おきの天気と服装指数、コーディネート例

### ユーザー管理
  - ユーザー登録
    - ユーザーの画像(任意)
    - 名前（必須）
    - emailアドレス（必須）
    - パスワード（必須）
    - パスワード確認（必須）
    - 都道府県設定(任意：デフォルトとして東京に設定、選択式（４７都道府県）)
    - 追加都道府県設定(任意：デフォルトとして大阪に設定、選択式（４７都道府県）)
    - 体感設定（任意：選択式 暑がり・やや暑がり・標準・やや寒がり・寒がり）
    #### LINEアカウントでも登録可能
  - ログイン
    - emailアドレス
    - パスワード
    #### Lineログインでも可能とする
  - ユーザー編集
    - ユーザーの画像(任意)
    - 名前（必須）
    - 都道府県設定(任意：デフォルトとして東京に設定、選択式（４７都道府県）)
    - 追加都道府県設定(任意：デフォルトとして大阪に設定、選択式（４７都道府県）)
    - 体感設定（任意：選択式 暑がり・やや暑がり・標準・やや寒がり・寒がり）
    - アカウント削除機能

### 服管理
  - 服一覧 
    - 名前
    - カテゴリーとサブカテゴリー
    - 季節
  - 服の登録
    - 服の画像（任意 画像もしくはデフォルトのアイコン）
     （CarrierWave、MiniMagickを実装）
    - 服の名前（必須）
      - 名前の自動入力（Google Cloud Vision APIおよびDeepL APIを使用して服の名前を自動入力）
      - 名前の手入力(string型にしてフリー入力)
    - 服のカテゴリー（必須）
    - 服のサブカテゴリー（必須 カテゴリーで項目を選択後、サブカテゴリーの項目を選択可能）
    - 服の備考欄（任意）
      - 色
      - 季節
      - 購入日
      - サイズ
      - 購入場所
      - 値段
      - 使用頻度
      - その他コメント(補足事項)
  - 服の編集
    - 登録した服の内容変更
    - 削除機能
- Line天気確認機能
  #### Line botに天気予報と入力した際に、登録した都道府県の気温・湿度・風速・天気のアイコンが表示される
- 問い合わせ 
  #### 特に体感温度につていて数値など
